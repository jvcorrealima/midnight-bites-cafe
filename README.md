# 🔪 Midnight Bites Cafe ☕

Bem-vindo à **Midnight Bites**, uma cafeteria imersiva inspirada nos clássicos filmes *slasher* dos anos 80. Este projeto foi desenvolvido como um desafio prático de Front-End, simulando um briefing real de um cliente (Ricardo Munhoz) que buscava uma identidade visual "dark" e funcional.

---

## 🚀 Tecnologias Utilizadas

O projeto foi construído focando em fundamentos sólidos de desenvolvimento web:

*   **HTML5:** Estruturação semântica da página.
*   **CSS3:** Estilização personalizada, uso de variáveis (`:root`), Flexbox para alinhamento e efeitos de transição.
*   **Google Fonts:** Integração das fontes *Creepster* (horror) e *Montserrat* (leitura).
*   **Responsividade:** Design pensado para uma boa visualização em diferentes dispositivos.

---

## 🛠️ Funcionalidades

- [x] **Hero Section:** Frase de impacto e Call to Action (CTA) para o cardápio.
- [x] **Sobre Nós:** Seção com storytelling baseado na estética VHS e filmes de terror.
- [x] **Cardápio "Crimes Famosos":** Grid de produtos com descrições detalhadas.
- [x] **Formulário de Vítimas:** Sistema de captura de leads (nome, idade, telefone) com validação de dados via HTML5.
- [x] **Interatividade:** Efeitos de *hover* e *active* nos botões para melhor UX (User Experience).

---

## 🎨 Identidade Visual

A paleta de cores foi escolhida para transmitir mistério e intensidade:
- **Background:** `#1a1a1a` (Preto Fosco)
- **Destaques:** `#8b0000` (Vermelho Sangue)
- **Texto:** `#f5f5f5` (Off-white para leitura confortável)
